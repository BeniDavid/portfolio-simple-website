# 🌐 Personal Portfolio Website

This is my personal portfolio website built using **HTML** and **CSS**. It showcases basic information about me and provides a way to get in touch.

---

## 📌 Features

* 🏠 Home / Info page with personal details
* 📬 Contact page
* 🎨 Simple and clean UI using CSS
* 📱 Beginner-friendly structure

---

## 🛠️ Built With

* HTML5
* CSS3

---

## 📂 Project Structure

```
portfolio/
│── index.html        # Main info page
│── contact.html      # Contact page
│── style.css         # Stylesheet
```

---

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
